docs/ux/storefront-landing-page.md — Section 4 — Shop by Sport (layout, sport tile dimensions, colours, typography, ambiguity resolution A-05, component states for tile and SHOP button, responsive behaviour)

## Description
Build the `ShopBySportSection` component: a 3-column (max) grid of sport category tiles. Each tile is a full-cover photograph with a dark overlay band at the bottom, a sport label, and a SHOP button. Both the tile and the SHOP button navigate to the filtered product listing. Content is dynamic (CMS-driven via `useSportCategories`). The tile count is flexible: 1, 2, or 3 tiles render gracefully without empty placeholders (UX A-05).

## Acceptance Criteria
- [ ] `src/components/ShopBySportSection.tsx` created as a named export
- [ ] Section heading "SHOP BY SPORT" — `text-xl`, `font-bold`, `uppercase`, `letter-spacing-wide`, `color-neutral-900`
- [ ] Tile grid: up to 3-column CSS grid, `gap: 0` (flush); responsive: 3-column lg, 3-column md (narrower tiles), 1-column sm
- [ ] 1 tile → `grid-cols-1` (full width); 2 tiles → `grid-cols-2` (50/50); 3 tiles → `grid-cols-3` (equal thirds)
- [ ] `SportTile` sub-component (can be internal to `ShopBySportSection.tsx`): relative-positioned container with `overflow: hidden`
- [ ] Tile height: `360px` (lg), `280px` (md), `200px` (sm)
- [ ] Background image: `object-fit: cover`, `object-position: center`; `loading="lazy"`
- [ ] Overlay band: `height: 80px`, `rgba(0,0,0,0.30)`, positioned `bottom: 0`
- [ ] Sport label: `bottom: 40px`, `left: 16px`, `color-brand-white`, `text-base`, `font-bold`, `uppercase`, `letter-spacing-wide`
- [ ] SHOP button: `bottom: 12px`, `left: 16px`, white bg / black text, `text-xs`, `font-bold`, `uppercase`, `letter-spacing-wide`, padding `6px 16px`, `radius-none`; `aria-hidden="true"` (parent `<a>` already describes the action)
- [ ] Entire tile wrapped in `<a aria-label="Shop [Sport name] shoes">` routing to `/products?sport=[sportSlug]`
- [ ] SHOP button click also routes to `/products?sport=[sportSlug]` (same destination as tile link) — use `e.stopPropagation()` to prevent double-navigation events
- [ ] Tile hover: overlay band darkens to `rgba(0,0,0,0.45)`, image scales `1.02`, `transition-base`; overflow-hidden clips the scale
- [ ] SHOP button hover: background `color-neutral-300`, `transition-fast`
- [ ] Focus on tile: `border-focus` outline `2px solid #111111`
- [ ] Loading state: skeleton tiles at correct height
- [ ] Section wrapped in `<section aria-label="Shop by Sport">`
- [ ] Section padding: `48px` top/bottom, `40px` horizontal (lg); reduced on sm per UX spec
- [ ] `prefers-reduced-motion`: scale transform on tile hover is removed; transition durations 0ms

## Technical Notes
- Dynamic column count: use a computed Tailwind class string based on `sportCategories.length`; ensure Tailwind's JIT includes all three variants by listing them in `safelist` in `tailwind.config.ts` if not used statically elsewhere
- The tile image `alt` text: use `[sportLabel] shoes` (e.g. "Running shoes")
- Keep the tile as an internal component (`SportTile`) inside the section file if the total stays under 300 lines; otherwise extract to `src/components/SportTile.tsx`
