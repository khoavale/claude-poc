## Description
Build the `Footer` component: a full-bleed dark footer with four content columns (Brand, Get Help, About, Social icons) and a legal bar below. Social platform URLs and footer link destinations are configuration-driven (not hardcoded) per UX A-08. The footer stacks to a single column on mobile.

## Acceptance Criteria
- [ ] `src/components/Footer.tsx` created as a named export
- [ ] Full bleed (`100vw`), background `color-footer-bg` (`#111111`), top/bottom padding `48px`
- [ ] Inner container: max-width `1200px` centred, horizontal padding `40px`
- [ ] Four-column flex layout, `gap: 48px`, `align-items: flex-start`:
  - Column 1 (Brand, `flex: 1.5`): logo wordmark (`font-black`, `16px`, white), links: Find a Store, Become a Member, Send Us Feedback
  - Column 2 (Get Help, `flex: 1`): heading "GET HELP", links: Order Status, Shipping & Delivery, Returns, Payment Options
  - Column 3 (About, `flex: 1`): heading "ABOUT NIKE", links: News, Careers, Investors, Sustainability
  - Column 4 (Social, `flex: 0 0 auto`): YouTube, Instagram, X (Twitter) icons ŌĆö `16├Ś16px`, white, `gap: 12px`; each icon in `<a target="_blank" rel="noopener noreferrer" aria-label="[Platform name]">`
- [ ] Column headings: `text-sm`, `font-bold`, `letter-spacing-wide`, `uppercase`, `color-footer-text`, margin-bottom `12px`
- [ ] Footer links: `text-sm`, `font-normal`, `color-footer-link` (`#CCCCCC`), `line-height: 28px`; hover ŌåÆ `color-footer-link-hover` (`#FFFFFF`), underline; focus ŌåÆ `border-focus` outline `2px solid #111111`
- [ ] Social icon URLs loaded from a config object `src/config/social.ts` exporting `{ youtube: string, instagram: string, twitter: string }` ŌĆö values are environment variables or defaults; no hardcoded profile URLs
- [ ] Footer link destinations defined in a `src/config/footerLinks.ts` config object ŌĆö no hardcoded strings scattered in the component; external links open `target="_blank" rel="noopener noreferrer"`
- [ ] Legal bar: background `color-footer-legal-bg` (`#000000`), height `48px`, flex layout `justify-content: space-between`; contains: region label "UNITED STATES", copyright notice, legal links (Guides, Terms of Sale, Terms of Use, Privacy Policy) at `gap: 16px`
- [ ] Legal bar text: `text-xs`, `color-footer-legal-text` (`#999999`)
- [ ] Footer wrapped in `<footer aria-label="Site footer">`, legal bar in `<div role="contentinfo">`
- [ ] Responsive ŌĆö mobile (sm): flex `column`; columns stack vertically; each column full-width; column heading `padding-top: 24px`; legal bar wraps to two rows
- [ ] Responsive ŌĆö tablet (md): `flex-wrap: wrap`; columns 1+2 on first row, columns 3+Social on second row; each `calc(50% - 24px)` wide

## Technical Notes
- SVG icons for social platforms: use a minimal inline SVG or an icon library already in the project ŌĆö do not introduce a new icon library just for this ticket. If none exists, create `src/components/icons/SocialIcon.tsx` with three inline SVGs as named exports
- The `<footer>` element itself is `role="contentinfo"` by default in HTML5; the inner legal bar `<div>` can omit the redundant `role` or use `aria-label` for differentiation ŌĆö check with QA
- Keep `Footer.tsx` under 300 lines; if the column link lists are verbose, extract `FooterLinkColumn.tsx`
